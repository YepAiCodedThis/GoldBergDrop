When you want more

Optional extras around the same tool — core flow is still drop → match → set up.

Workshop

Queue Steam Workshop items and land them in the right folder.

  • Paste links or IDs into a download queue
  • GGNetwork first, SteamCMD as fallback
  • Output folder under Settings

GreenLuma

Optional Steam helper — install and control from one place.

  • Bundled install + CSF / pack import
  • AppList sync and DLLInjector
  • Start Steam with or without GreenLuma
  • Download prompts wait until you leave the game

Tray launcher

Keep games one right-click away after setup.

  • Launch tracked titles from the tray menu
  • Close the window to the notification area
  • Autostart with Windows if you want

Explorer Send to

Ship an exe to GoldbergDrop without opening the window first.

  • Enable once in Settings
  • Right-click → Send to → GoldbergDrop
  • Same drop → match → set up flow

Single instance

No pile of windows when you open another file.

  • Second launch focuses the running app
  • Extra drops hand off via IPC
  • Tray stays the one home base

Needs & limits

Small checklist so nothing surprises you later.

  • Windows · one portable exe · no login
  • 7-Zip only for GreenLuma / archives
  • For games you own — no games in the download

Tip: GreenLuma is great for play, rough for updates. Prefer plain Steam for downloads — GoldbergDrop can remind you after you exit a game.
